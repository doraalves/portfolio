# 🖥️ Izadora Alves — Portfolio

> Portfólio pessoal desenvolvido com React e Styled Components.

**🔗 [doraalves.github.io/portfolio](https://doraalves.github.io/portfolio)**

---

## 🚀 Tecnologias

- [React](https://react.dev/) — biblioteca de interface
- [Styled Components](https://styled-components.com/) — estilização com CSS-in-JS
- [EmailJS](https://www.emailjs.com/) — envio de emails pelo formulário de contato
- [GitHub Pages](https://pages.github.com/) — hospedagem

---

## 📁 Estrutura

```
src/
├── components/
│   ├── Icons.js            # SVG icons
│   ├── Modal.js            # Modal de detalhes dos projetos
│   ├── Navbar.js           # Navegação fixa
│   └── ParticleCanvas.js   # Canvas interativo com partículas
├── data/
│   └── projects.js         # Dados dos projetos
├── pages/
│   ├── About.js            # Sobre + experiência + ilustrações
│   ├── Contact.js          # Contato com formulário
│   ├── Home.js             # Página inicial com canvas
│   └── Projects.js         # Projetos
└── styles/
    ├── GlobalStyle.js
    ├── About.styled.js
    ├── Contact.styled.js
    ├── Home.styled.js
    ├── Modal.styled.js
    ├── Navbar.styled.js
    └── Projects.styled.js
```

---

## ⚙️ Rodando localmente

**Pré-requisitos:** Node.js 16+

```bash
# Clonar o repositório
git clone https://github.com/doraalves/portfolio.git

# Entrar na pasta
cd portfolio

# Instalar dependências
npm install

# Rodar em desenvolvimento
npm start
```

O projeto abre em `http://localhost:3000`.

---

## 📧 Configurando o EmailJS

O formulário de contato usa o [EmailJS](https://www.emailjs.com/) para enviar mensagens direto para o email.


---

## 🌐 Deploy no GitHub Pages

```bash
# Instalar gh-pages (apenas na primeira vez)
npm install gh-pages --save-dev

# Fazer deploy
npm run deploy
